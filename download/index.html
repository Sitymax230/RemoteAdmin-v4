⚡ Возможности

12 ключевых функций

Полнофункциональная система для удалённого администрирования.

💻

Удалённый рабочий стол

Скриншоты, трансляция экрана в реальном времени, управление мышью и клавиатурой удалённого ПК.

📈

Мониторинг в реальном времени

CPU, RAM, использование диска через WebSocket. Обновление каждую секунду без перезагрузки страницы.

🛒

App Store

Магазин приложений с локальной установкой с сервера. Загрузка APK/EXE и автоматическое развёртывание.

🎫

Тикеты

Система заявок: пользователи создают тикеты с описанием проблемы, администраторы решают и закрывают.

🔐

2FA аутентификация

Двухфакторная аутентификация TOTP с QR-кодом для настройки через Google Authenticator.

👥

Управление пользователями

Ролевая модель: superadmin, admin, viewer. Гибкое управление правами доступа к системе.

🖥

Заставка

Экран «Обновление системы» — блокировка рабочего стола пользователя на время обслуживания.

📊

Дашборды

Графики метрик, статистика по агентам, аналитика использования и мониторинг системы.

🔧

Терминал

Удалённое выполнение команд на агентах. Вывод stdout/stderr в реальном времени через WebSocket.

🔄

Автообновления

Удалённое обновление агента с сервера. Автоматическая проверка версии и бесшумная установка.

👁️

Стелс-режим

Скрытый запуск агента без консольного окна. BAT и VBS лаунчеры для невидимой работы.

📁

Файловый менеджер

Браузер файлов на сервере. Навигация по директориям, просмотр, загрузка и скачивание файлов.

⚡ Быстрый старт

Установка и запуск

Три шага для запуска сервера и подключения агентов.

bash 
# 1. Перейдите в папку проекта и установите зависимости
$ cd RemoteAdmin-v4
$ npm install

# 2. Инициализируйте базу данных
$ npm run db:push

# 3. Заполните демонстрационными данными
$ npm run db:seed

# 4. Запустите сервер разработки
$ npm run dev

# Сервер будет доступен на http://localhost:3000
# Логин: admin / Пароль: admin123
powershell / cmd 
# 1. Скопируйте папку standalone/ на целевой компьютер
> xcopy /E /I standalone C:\RemoteAdmin\agent

# 2. Установите зависимости (только для Node.js версии)
> cd C:\RemoteAdmin\agent
> npm install --production

# 3. Запустите агент
> node agent.js --server ws://YOUR_SERVER:3000

# Или используйте готовый EXE (без Node.js)
> remoteadmin-agent-win.exe --server ws://YOUR_SERVER:3000

# Скрытый запуск (без окна консоли)
> start-hidden.bat
bash 
# 1. Установите Node.js (если нужно для исходной версии)
$ curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
$ sudo apt-get install -y nodejs

# 2. Скопируйте папку standalone/
$ cp -r standalone/ ~/remoteadmin-agent
$ cd ~/remoteadmin-agent

# 3. Установите зависимости (для Node.js версии)
$ npm install --production

# 4. Запустите агент
$ node agent.js --server ws://YOUR_SERVER:3000

# Или используйте готовый бинарник (без Node.js)
$ chmod +x remoteadmin-agent-linux
$ ./remoteadmin-agent-linux --server ws://YOUR_SERVER:3000
